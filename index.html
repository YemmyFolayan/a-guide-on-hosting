Steps To Host Your Files Using Github Pages

This is a short guide on hosting your frontend files using Github pages

Create a Github account Click here
And if you have created an account already, log in and follow the steps

Create a repository

A page will come up where you'll fill the name of the repository then locate the checkbox with the text Initialize README beside it and check it, then create

Then the page of the "dashboard" of that repository will come up and you'll see an upload files button click it and upload all your files

So after your repo have been successfully created, you go to settings

Scroll down and locate Github Pages, then you will see source
Click the dropdown box and select master branch

The page will reload, when it does, you'll see a link just under the Github Pages, copy the link

Scroll up and locate Code in the nav bar, click on it

You will see the files you have uploaded, select the README.md file

When it loads, you will see a pencil icon, click on it and paste the link you previously copied, then you commit changes

After your file is saved, copy the link in the URL bar, that is what you will be submitting
